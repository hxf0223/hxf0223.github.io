# 通过函数指针地址找到函数

```bash
# 通过地址找到函数声明
info symbol 0x7f0db14cf57e
# 通过地址找到函数在哪一行
info line *0x7f0db14cf57e
# 查看加载了哪些共享库
info sharedlibrary
```
