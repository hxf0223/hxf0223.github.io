## 1. Markdown 转义字符列表

使用反斜杠（"\\"）可以转义 Markdown 中的特殊字符，使其被当作普通文本显示。以下是 Markdown 中的特殊字符列表：

```text
\  backslash
`  backtick
*  asterisk
_  underscore
{} curly braces
[] square brackets
() parentheses
#  hash mark
+  plus sign
-  minus sign (hyphen)
.  dot
!  exclamation mark
```

理论上， "\\" 可以转义任何字符，如果该字符不是特殊字符也会原样输出。这里的理论上，其实说的是转义直接出现在正文中。

## 2. 在 Markdown 中插入特殊符号

### 2.1 任务列表符号

| 符号       | 用途               |
| ---------- | ------------------ |
| ✅ ☑ ✓ ✔ √ | 完成、正确、同意   |
| ❌ ☒ ✘ ✕ ✖ | 未完成、错误、否决 |
| ⏳ ⌛      | 等待、进行中       |

### 2.2 状态与进度

| 类别   | 符号           | 说明                   |
| ------ | -------------- | ---------------------- |
| 进度   | ⏳ ⌛ ✅ 🚧 🔄 | 等待、完成、进行、循环 |
| 优先级 | 🔴 🟡 🟢       | 高、中、低             |
| 标记   | 📌 📍 🔖       | 重要、位置、书签       |

### 2.3 工具与操作

| 符号     | 用途               |
| -------- | ------------------ |
| 🔧 ⚙️ 🔩 | 工具、设置、配置   |
| 🔍 🔎    | 搜索、查看、浏览   |
| 📁 📂 📋 | 文件夹、文件、文档 |
| 🔗 ⛓️    | 链接、连接、关联   |

### 2.4 提示与注意

| 符号  | 用途             |
| ----- | ---------------- |
| 📢 📣 | 通知、宣布、重要 |
| ⚠️ 🚨 | 警告、注意、危险 |
| 💡 💭 | 建议、想法、思考 |
| ℹ️ 📝 | 信息、说明、备注 |
| 🔍 📊 | 分析、详解、查看 |

### 2.5 美化装饰符号

| 符号              | 说明             |
| ----------------- | ---------------- |
| 🌹 🍀 🌙 🍂 🍃 🌷 | 自然、植物       |
| 💎 🔥 ⭐ 🍄 🏆    | 宝贵、热门、优秀 |
| ↔️ ➡️ ⬅️ ⬆️ ⬇️    | 方向箭头         |
| ✨ 🌟 💫          | 闪耀、高亮       |
